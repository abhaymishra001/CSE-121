📊 Unit 1: Data Science & Big Data
  • Data Science: Interdisciplinary field extracting knowledge from structured and unstructured data using scientific methods, algorithms, and systems.
  • Job Roles: Data Scientist (builds models), Data Analyst (finds trends/reports), Data Engineer (builds data pipelines/infrastructure).
  • Big Data: Data so large and complex that traditional data processing tools can't handle it.
  • The 5 V's of Big Data: Volume (scale/size), Velocity (speed of generation), Variety (types/formats), Veracity (accuracy/quality), Value (usefulness).
  • Hadoop Ecosystem: Open-source framework for distributed storage & processing.
  • Hadoop Components: HDFS (Storage across clusters), MapReduce (Processing parallelly), YARN (Resource Management).
  • Other Big Data Tools: Hive (SQL-like querying), Pig (Scripting), HBase (NoSQL Database), Spark (In-memory processing, 100x faster).
  • Data Visualization: Graphical representation of data to spot patterns/trends.
  • Tableau: Leading visual analytics platform. Drag-and-drop interface requiring no coding skills.
  • R Language: Open-source programming language for statistical computing and graphics. Free under GNU GPL license.
🤖 Unit 2: Artificial Intelligence & ML
  • Artificial Intelligence (AI): Simulation of human intelligence by machines. Allows machines to think, reason, and solve problems.
  • Types of AI: Narrow AI (specific tasks like Siri), General AI (human-level cognition), Super AI (surpasses human intelligence).
  • Machine Learning (ML): A subset of AI where machines learn from data and improve over time without being explicitly programmed.
  • Supervised Learning: Model is trained on labeled data (inputs with known outputs). Examples: Classification (spam detection), Regression (price prediction).
  • Unsupervised Learning: Model is trained on unlabeled data. Finds hidden patterns. Example: Clustering (customer segmentation).
  • Reinforcement Learning: Agent learns by trial-and-error using rewards and penalties (e.g., self-driving cars, chess AI).
  • Deep Learning: Subset of ML using Artificial Neural Networks (ANN) with many layers. Mimics the human brain structure. Best for image and speech recognition.
  • NLP (Natural Language Processing): Enables computers to understand, interpret, and generate human language. Examples: Chatbots, Google Translate, Sentiment Analysis.
  • Computer Vision: Enables machines to interpret visual information from the world (images/video). Examples: Facial recognition, medical image analysis.
  • Expert Systems: AI programs that emulate the decision-making ability of a human expert using a knowledge base and inference engine.
🛡️ Unit 3: Cybersecurity
  • Cybersecurity: Practice of protecting systems, networks, and data from digital attacks, theft, or damage.
  • CIA Triad: The foundation of security. Confidentiality (encryption/access limits), Integrity (hashing/preventing tampering), Availability (backups/redundancy).
  • Malware Types: Virus (attaches to files), Worm (self-replicates over networks), Trojan (disguised as legit software), Spyware (monitors activity secretly), Ransomware (encrypts data for money).
  • Phishing: Social engineering attack using fake emails/messages to steal credentials. "Spear phishing" targets specific individuals.
  • DoS & DDoS Attacks: Denial of Service. Overwhelming a server with traffic so legitimate users cannot access it. Distributed (DDoS) uses botnets.
  • Brute Force: Systematic trial-and-error guessing of passwords. Prevented by strong passwords and account lockouts.
  • Zero-Day Attack: Exploit of an unknown vulnerability with no available patch ("zero days" to fix).
  • Secure Browsing: Look for HTTPS (encrypted connection) and a padlock icon. Avoid suspicious links.
  • Digital Footprint: Trace of online activities. Active (posts/comments you make) vs Passive (cookies/tracking without input).
  • Security Tools: Nmap (Network scanning & port discovery), Wireshark (Packet sniffing & traffic analysis), AI-based detection.
  • Compliance: Adhering to laws. Examples: ISO 27001 (ISMS), GDPR (EU privacy), IT Act 2000 (India).
🔄 Unit 4: DevOps & Software Testing
  • DevOps: Development (Dev) + Operations (Ops). Culture and practices emphasizing collaboration, automation, and CI/CD for rapid delivery.
  • DevOps Lifecycle: 8 phases: Plan → Develop → Build → Test → Release → Deploy → Operate → Monitor. A continuous cycle.
  • CI/CD: Continuous Integration (auto-merging & testing code daily). Continuous Delivery (code is always deployable, manual approval). Continuous Deployment (fully automated release to production).
  • DevOps Tools: Git (Version Control), Docker (Containerization), Kubernetes (Container Orchestration), Jenkins (CI/CD server), Maven (Build tool), Ansible/Puppet (Config management).
  • Software Testing: Verifying a software application is bug-free, meets requirements, and satisfies users.
  • Testing Types: Black Box (testing inputs/outputs without code knowledge). White Box (testing internal logic/code structure). Gray Box (hybrid).
  • Testing Levels: Unit (individual components) → Integration (modules combined) → System (complete system) → Acceptance (end-user validation).
  • Manual vs Automation: Manual (human testers, good for UI). Automation (scripts/tools like Selenium, good for repetitive/regression testing).
  • Test Case Design: Includes ID, description, steps, expected result. Methods: Boundary Value Analysis (testing edges), Equivalence Partitioning.
  • Defect Life Cycle: New → Assigned → Open → Fixed → Retest → Verified → Closed. (Deferred = postponed, Rejected = not a bug).
☁️ Unit 5: Cloud Computing
  • Cloud Computing: On-demand delivery of computing services (servers, storage, databases, networking) over the internet with pay-as-you-go pricing.
  • Key Characteristics: Broad network access, Resource pooling, Rapid elasticity (scaling up/down), Measured service.
  • IaaS (Infrastructure as a Service): Renting IT infrastructure (servers, VMs, storage). You manage the OS and apps. Example: AWS EC2, Azure VMs.
  • PaaS (Platform as a Service): Provides an environment/framework for developers to build apps without managing underlying infrastructure. Example: Heroku, Google App Engine.
  • SaaS (Software as a Service): Delivering software applications over the internet on a subscription basis. You just use the app. Example: Gmail, Salesforce.
  • Public Cloud: Owned by third-party providers (AWS, Google). Shared resources, cost-effective, scalable.
  • Private Cloud: Used exclusively by one organization. High security, maintained on a private network.
  • Hybrid Cloud: Combines public and private clouds, allowing data to be shared between them for flexibility.
  • Virtualization: Creating virtual versions of computing resources using a Hypervisor.
  • Hypervisor Types: Type 1 (Bare Metal): Runs directly on hardware (e.g., VMware ESXi). Type 2 (Hosted): Runs on top of a host OS (e.g., VirtualBox, VMware Workstation).
🌐 Unit 6: Web Dev & UI/UX
  • Web Development: Building/maintaining websites. Consists of Frontend (client-side), Backend (server-side), and Full Stack (both).
  • UI (User Interface): The visual part. Focuses on aesthetics, layout, buttons, interactivity. Types: GUI (Graphical), VUI (Voice), MDI (Menu-driven).
  • UX (User Experience): The feel and journey. Focuses on usability, logic, ease of use, and overall user satisfaction.
  • Frontend Trinity: HTML (Skeleton/Structure), CSS (Styling/Appearance), JavaScript (Behavior/Interactivity).
  • Backend: Server-side logic & database interactions. Technologies: PHP, Node.js, Python, Java.
  • PHP: Server-side scripting language (.php files). Executes on server, returns HTML to browser. Open-source.
  • CRUD Operations: The 4 basic database functions: Create, Read, Update, Delete.
  • Responsive Design: Adapting layout to screen sizes using CSS Media Queries. "Mobile-first" = design for smallest screens first.
  • SPA (Single Page Application): Rewrites current page dynamically rather than loading new entire pages (e.g., Gmail, Netflix). Faster UX.
  • AR (Augmented Reality): Overlays digital objects onto the real world (partially immersive). Uses SLAM (Simultaneous Localization & Mapping). E.g., Pokémon GO.
  • VR (Virtual Reality): Creates a completely simulated 3D environment replacing reality (fully immersive). Requires VR headsets.
  • Tools & IDEs: VS Code (popular code editor), Unity/Unreal Engine (AR/VR dev), Figma (UI/UX design).
